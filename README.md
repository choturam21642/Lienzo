# 🎨 Lienzo - AI Image Generation Platform

Lienzo is a full-stack MERN application that allows users to generate high-quality images using AI. It features a modern UI, secure authentication, and a credits-based system for image generation.

---

## 🚀 Features
* **AI Image Generation:** Convert text prompts into stunning visuals.
* **User Authentication:** Secure Sign-up/Login functionality.
* **Credit System:** Users can track and use credits for generating images.
* **Responsive Design:** Fully optimized for Mobile and Desktop.
* **MERN Stack:** Built with MongoDB, Express, React, and Node.js.

---

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
